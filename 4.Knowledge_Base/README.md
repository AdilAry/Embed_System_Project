#### Shortcuts Setup;
---
ctrl + d -> toggle viewing and editing mode. 
ctrl + s -> close current tab.

ctrl + w -> advanced search.

ctrl + tab -> move between tabs.

ctrl + g -> open a graph view.

ctrl + L -> delete current file.
